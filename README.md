# How to make it work
Firstly get Turbowarp on the Microsoft Store On Windows, Flathub on linux, Via the web on MacOS.

Download the release on github, and put it in to turbowarp.

Head to the costumes tab and start creating shapes using the circle and square and make one that looks like a flower!
